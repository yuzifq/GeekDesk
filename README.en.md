A small, **beautiful** desktop quick-launch management tool with integrated Everything search.

## Integrated Everything Search for Fast Full-Disk File Searches

- Everything is integrated in version 2.5.14 and later.

![](https://s3.bmp.ovh/imgs/2023/04/25/ecfbf59916b9b854.gif)

## Global Hotkeys, One-Key Launch, and Mouse Following

- Configure custom hotkeys and use the shortcuts that suit your workflow.
- Launch the panel with one key, or use the middle mouse button.
- Enable mouse following to automatically track the cursor position.
